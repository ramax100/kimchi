# Kimchi Terminal

Web terminal untuk menjalankan Kimchi Coding agent di browser.

## Deploy di Render.com (Gratis)

1. Buka https://render.com → sign up pakai GitHub
2. New → Web Service → pilih repo `ramax100/kimchi`
3. Build Command: `npm install`
4. Start Command: `npm start`
5. Instance: Free
6. Deploy!

## Cara Pakai

1. Tap **⬇️ Install** → tunggu selesai
2. Tap **🔑 Key** → paste API key dari app.kimchi.dev
3. Tap **🥬 Kimchi** → agent mulai
4. Jika muncul pertanyaan Yes/No → tap tombol **Yes** atau **No**
5. Jika muncul pilihan 1/2 → tap tombol **1** atau **2**
6. Tap **💬 Perintah** → ketik tugas coding

## Tombol

| Tombol | Fungsi |
|--------|--------|
| 🥬 Kimchi | Jalankan agent |
| Yes / No | Jawab prompt |
| 1 / 2 | Pilih opsi |
| 💬 Perintah | Kirim tugas ke agent |
| 🧪 Ferment | Mode otonom |
| ⬇️ Install | Install Kimchi CLI |
| 🔑 Key | Set API key |
| Clear | Bersihkan layar |

## Lokal

```bash
npm install
npm start
# Buka http://localhost:3000
```
