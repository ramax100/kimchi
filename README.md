# Kimchi Terminal

Web-based terminal emulator inspired by [Kimchi Coding](https://kimchi.dev).

## Usage

Open `index.html` in any browser. No dependencies needed.

## Commands

- `help` - Show commands
- `kimchi` - Start session
- `ls`, `cd`, `cat`, `pwd` - File navigation
- `/ferment` - Autonomous workflow
- `/help` - Slash commands
